# Portofolio Adi (Sederhana)

Ini adalah situs portofolio sederhana untuk Adi. Struktur file:

- `index.html` — Halaman utama (beranda, tentang, proyek singkat)
- `layanan.html` — Daftar layanan
- `kontak.html` — Form kontak
- `css/style.css` — Stylesheet utama

Cara menjalankan:

1. Buka file `index.html` di browser (cukup double-click file atau jalankan server lokal).
2. Untuk melihat perubahan stylesheet, edit `css/style.css` dan refresh browser.

Catatan: Form pada `kontak.html` saat ini tidak mengirim data ke backend — perlu integrasi server atau layanan form pihak ketiga.
